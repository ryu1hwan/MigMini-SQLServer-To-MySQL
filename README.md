# MigMini-SQLServer-To-MySQL
MS SQL Server to MySQL, 데이터 마이그레이션 프로그램입니다.
  - 데이터만 마이그합니다.
  - SQL 구문 마이그는 아닙니다.

실행을 위해 필요한 환경
  - py소스 밑에 csv_res 라는 빈폴더를 생성해주세요.
  - (그래야, 결과를 csv로 내보낼 수 있습니다.)
  - 아래의 모듈 install이 필요합니다.
  - pip install pymssql
  - pip install pymysql
  - pip install pandas
  - pip install PyQt5
